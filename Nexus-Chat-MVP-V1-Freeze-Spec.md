# Nexus Chat (极速社交助手) - V1 冻结规范

## 【核心生存链路 · Golden Loop】

[用户认证] -> [选择联系人] -> [发送文本消息]

## 【V1 冻结决议】

V1 仅支持以下功能：
1. 邮箱注册与登录
2. 查看联系人列表
3. 发送和接收纯文本消息
4. 消息持久化存储（单表结构）

V1 不做任何其他事。

## 【DEFERRED 清单】

- 密码重置与账户验证
- 用户资料管理（昵称、头像）
- 高清图片消息（支持压缩与预览）
- 语音消息（支持录制与播放）
- 消息状态显示（发送中、已发送、已送达、已读）
- 群组聊天（创建与管理群组、群组消息发送与接收、群组成员管理）
- 消息搜索功能
- WebSocket实时消息推送
- 消息订阅与增量更新
- 在线状态显示
- 消息加密与用户数据安全保护
- 内容审核机制
- 数据导出和删除功能
- 云存储方案
- 消息队列
- 第三方实时通讯服务
